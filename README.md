# 한장 — 원터치 카드뉴스 스튜디오

주제 한 줄을 입력하면 카드뉴스 전체 흐름을 만들고, 각 카드의 문구·역할·스타일·브랜드를 바로 수정할 수 있는 웹 앱입니다.

## 가장 빠르게 실행하기

Windows에서 `start-app.cmd`를 더블클릭하세요. 로컬 전용 서버가 실행되고 브라우저에서 `http://127.0.0.1:4173`이 열립니다.

직접 실행하려면 다음 명령을 사용합니다.

```powershell
node serve.mjs
```

앱은 외부 API나 환경변수 없이 동작합니다. `index.html`과 `standalone.js`가 의존성 없는 실행본입니다.

## 주요 기능

- 주제·원문·독자·말투·카드 수를 바탕으로 5~10장 원터치 생성
- 표지, 문제 제기, 인사이트, 단계, 인용, 체크리스트, 요약, CTA 역할 기반 구성
- 캔버스 직접 편집과 오른쪽 속성 패널 편집
- 카드 복제·삭제·추가·드래그 재정렬
- 카드별 `더 짧게`, `훅 강화`, `친근하게` 부분 재작성
- 6가지 디자인 테마와 문구 보존형 전체 스타일 교체
- 브랜드 이름·로고 글자·포인트 색상·푸터 일괄 적용
- 자동 저장, 실행 취소·다시 실행, 마지막 작업 이어하기
- 현재 카드 PNG, 전체 PNG ZIP, 브라우저 PDF 인쇄
- 캡션·해시태그 자동 조합 및 복사
- 1080×1350(4:5) 내보내기

## 프로젝트 구조

- `index.html`, `standalone.js`, `serve.mjs`: 바로 실행 가능한 무의존성 앱
- `src/components`: Next.js/React 편집 화면
- `src/lib/cardnews.ts`: 결정론적 카드뉴스 생성·부분 재작성 엔진
- `src/lib/export.ts`: SVG, PNG, ZIP, PDF 내보내기 엔진
- `src/app/globals.css`: 전체 디자인 시스템과 반응형 레이아웃

## Next.js 개발 모드

디스크 공간이 충분한 환경에서는 아래 명령으로 React 소스도 실행할 수 있습니다.

```powershell
npm install
npm run dev
```

현재 실행본은 AI API 비용과 키 없이도 즉시 테스트할 수 있도록 로컬 생성 엔진을 사용합니다. 실제 LLM API를 연결할 때는 `src/lib/cardnews.ts`의 반환 타입을 유지한 서버 라우트를 추가하면 편집기와 내보내기 흐름을 그대로 재사용할 수 있습니다.
